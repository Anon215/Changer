# Changer
Script, um automatisiert Profileinstellungen unter Windows vorzunehmen

Es werden zBsp Energieeinstellungen, Windows Updates, Registrykeys und
Einiges mehr geändert.

Bitte unbedingt den Code der Datei Vorinstallation_v1.0.bat anschauen und ggf anpassen!
Auch sollte das Logo durch etwas Schöneres ersetzt werden, wenn gewünscht.
